# unifecaf-flix-api
API REST em Node.js para a plataforma fictícia UniFECAF Flix. Exponde um catálogo de filmes armazenado em MySQL, seguindo arquitetura MVC e padrão REST. Possui endpoints para listar todos os filmes, buscar por ID e filtrar por nome/sinopse, com código organizado, tratamento de erros e uso correto de status HTTP.
